# Node.js Homework 
Welcome to LIRI. LIRI is like iPhone's SIRI. However, while SIRI is a Speech Interpretation and Recognition Interface, LIRI is a Language Interpretation and Recognition Interface. LIRI will be a command line node app that takes in parameters and gives you back data.

## About the APP
Liri has 4 commands 
1. 'my-tweets'
2. 'spotify-this-song'
3. 'movie-this'
4. 'do-what-it-says'

## How to use
Use node to run this app. Use the 4 commands provided by liri to access. Run all the commands through the console. 


## Built With
- Node.js (packages: twitter, spotify, request)
